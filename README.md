# System-biology
Repository which contain a series of models description and little projects which have the goals to try to describe real systems in a mathematical way.
General description of the notebooks(more details and explanation inside the notebooks):
* Numerical Analusis of ODEs: analyze and solve system of differential equations using an example model.
* Parameter Estimation: Estimates parameters of a model.
* Markov models: Implement a practictal application of Markov Model in order to predict the type of language of a given text.
* Car simulation: Implement a genetic algorithm in order to allow the evolution of randomly constructed cars with variables like: size of and position of weels, upper shape and lower shape. After n-generations of evolution select the car which complete a track in the lowest ammount of time.
* Hill function: Try to mathematically model how the activity of a promoter depends on the concentration of its active transcription factor.
* Aploid models of Natural Selections: theoretical notebook which try to describe in a discrete mathematical way how the alleles frequency in an aploid organisms evolve during the time.

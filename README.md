# System-biology
Repository which contain a series of excercise and little projects which have the goals to try to describe real systems in a mathematical way.
General description of the notebooks(more details and explanation inside the notebooks):
* Numerical Analusis of ODEs: analyze and solve system of differential equations using an example model.
* Parameter Estimation: Estimates parameters of a model.
* Markov models: Implement a practictal application of Markov Model in order to predict the type of language of a given text.
* Car simulation: Implement a genetic algorithm in order to allow the evolution of cars (e.g. size of weels, upper shape and lower shape) and selecting the best one after n-generations which complete a track in the lowest ammount of time.
